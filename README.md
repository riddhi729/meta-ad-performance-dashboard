# 📊 Meta Ad Performance Dashboard

A Power BI dashboard designed to analyze Facebook and Instagram ad performance, uncover audience behavior, and generate actionable insights for better marketing decisions.

---

## 🚀 Project Overview

This project focuses on analyzing Meta advertising data to evaluate campaign effectiveness across platforms.

The dashboard provides a clear and interactive view of key performance metrics, helping identify trends, optimize campaigns, and improve overall ROI.

---

## 🎯 Business Objective

- Analyze ad performance across Facebook and Instagram  
- Track key marketing metrics such as CTR, Engagement, and Conversion Rate  
- Understand audience behavior based on age, gender, and location  
- Identify high-performing ad types and campaigns  
- Enable data-driven decision making  

---

## 📁 Dataset Description

The project uses structured datasets representing different aspects of advertising data:

- ads.csv → Contains ad-level information  
- ad_events.csv → Tracks user interactions and engagement  
- campaigns.csv → Contains campaign-related details  
- users.csv → Includes user demographic data  

---

## 🔗 Data Relationships

- ads is connected to campaigns using campaign_id  
- ad_events is linked to ads using ad_id  
- users is linked to ad_events using user_id  

---

## 🛠️ Tools and Technologies

## 📊 Dashboard Features

### KPI Metrics
- Impressions  
- Clicks  
- Engagement  
- Purchases  
- CTR (Click-Through Rate)  
- Conversion Rate  

### Analysis Sections
- Audience analysis by gender  
- Audience analysis by age  
- Weekly performance trends  
- Hourly engagement patterns  
- Country-wise performance  
- Ad type comparison  

### Filters and Interactivity
- Platform selection (Facebook / Instagram)  
- Dynamic metric selection  
- Interactive visuals and drill-down  

---

## 💡 Key Insights

- Instagram shows higher engagement compared to Facebook  
- Younger audiences interact more with ads  
- Certain ad types drive better conversions  
- Engagement peaks at specific hours of the day
